# 🤖 SSV Telegram Bot

> Telegram-native DeFi Control Layer for Sentient Shield Vault (SSV)

---

## 🚀 Overview

SSV Telegram Bot is a real-time DeFi assistant that allows users to **monitor and control their vault directly from Telegram**.

It connects **Web2 (Telegram UX)** with **Web3 (on-chain data)**.

---

## ⚡ Features

### 🔗 Wallet Integration

* `/connect` → Link your wallet
* Persistent wallet storage

### 📊 Live Dashboard

* Deposit (on-chain)
* Rewards (real-time)
* XP & Level
* Bonus %

### 🔐 NexaID Integration

* Verification status
* Reputation score
* Reward bonus

### 🎁 Smart Alerts

* Auto reward notifications
* Triggered only once (no spam)

### 🛡️ Risk Detection

* Alerts for high exposure
* Suggests rebalancing

### 🏆 Gamification

* XP tracking
* Level system
* Leaderboard support

### 🚀 Deep Linking

* Open dApp directly from Telegram

## 🛠️ Setup

### 1️⃣ Clone Repo

git clone <your-repo>
cd ssv-telegram-bot

---

### 2️⃣ Install Dependencies

npm install

---

### 3️⃣ Setup Environment

Create `.env` file:

BOT_TOKEN=your_telegram_bot_token

---

### 4️⃣ Run Bot

node index.js

---

## 📡 Commands

* /start → Open control panel
* /connect → Link wallet
* /dashboard → View vault data
* /verify → NexaID status
* /leaderboard → Ranking
